## Typography 
 Headings (Bold Montserrat): 
 H1: 36–40px 
 H2: 28–24px 
 H3: 20–24px 
 Use font-weight: 700 for a bold, clear look. Adjust line height around 1.2 for clarity.elementor+2​ 
  
 Subheadings (Raleway): 
 H2/H3: 20–24px for main subheaders, 16–20px for lower-level subheaders 
 Use font-weight: 500–600. Line height: 1.3–1.4 for easy reading.fonts.google+2​ 
 Body (Source Sans Pro): 
  
 14–16px is optimal for web content 
 Use font-weight: 400. Line height: 1.5–1.7 to maintain readability for longer text.
